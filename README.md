# Cinema Guide
Cinema-Guide App is a simple iOS application,
that allows users to browse and discover various Movies. Users can explore different Movies categories,
view movie details, and create their own collection of favorite Movies.

## Used In The APP
- SWIFT
- Clean Architecture 
- Network Layer (URLSession)
- MVVM Architecture Pattern
- Local data persistance(ŸçRealm)
- Autolayout for portrait and landscape 
- RxSwift, RxCocoa For Data Binding
- XIB Files
